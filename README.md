Task 4-Basic Chatbot

📖Description

The Basic Chatbot is a simple rule-based chatbot developed using Python. It responds to predefined user inputs such as greetings, questions, and exit commands. The chatbot continues the conversation until the user types "bye".

🚀Features

- Responds to greetings
- Answers simple questions
- Continues chatting until the user exits
- Beginner-friendly
- Console-based application

🛠️Technologies Used

- Coding Python

📂 Project Structure
```
Basic_Chatbot/
│── chatbot.py
│── README.md
```

▶️How to Run

1. Open Pydroid 3 or any Python IDE.
2. Create a file named `chatbot.py`.
3. Copy and paste the Python code.
4. Save the file.
5. Run the program.
6. Type messages such as **hello**, **how are you**, or **bye**.

👇 Concepts Used

- Functions
- While Loop
- If-Elif-Else
- Input and Output
